      How to Play

      Press the … button to show additional controls


      Press the right arrow key or the ✔ button if you got it correct


      Press the down arrow key or the x button if you got it incorrect


      Press the left arrow key or the ½ button if you only got half of the answer


      Press the up arrow key or the ^ button to show the current song

      Adding Song Lists

      Press the + button to add extra song lists
      Song lists are saved on a session basis. This means your lists will be erased on a page reload,

      You can download generated JSON for future use.

      Important note: improperly formatted CSV, TSV or JSON will result in uncaught errors.
